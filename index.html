Sobre a exposição

Arte de Código Aberto propõe uma jornada no mundo da arte gerativa, onde o público é convidado a mergulhar na intersecção entre linguagem de programação e expressão artística. Esta exposição individual do artista-programador Vamoss nos mostra como os códigos abertos podem ser manipulados para criar imagens e ideias.

A mostra, que já encantou visitantes no espaço ACT do Sesc Tijuca em 2022 e na Sala de Ciências do Sesc Madureira entre junho e setembro de 2023, agora marca sua presença no Laboratório de Atividades do Amanhã. Aqui, os visitantes têm a chance de experimentar a arte de forma interativa, criando suas próprias obras generativas e acessando o dinamismo entre ordem e caos.

Esta exposição não é apenas uma celebração da criatividade; ela desafia os visitantes a refletirem sobre presença e influência dos programas de computador em nossas vidas - desde dispositivos móveis até mídias sociais, afetando nossas emoções e comportamentos. Arte de Código Aberto é focada na interatividade, convidando os visitantes a se envolverem ativamente com a arte. A exposição é desenhada para incentivar que o público se envolva com os códigos e ver suas próprias criações ganharem vida. Essa interação proporciona uma compreensão mais profunda da relação entre arte, tecnologia e a sociedade.

Sobre a artista

Vamoss, artista-programador e Diretor de Tecnologia na SuperUber, é reconhecido por sua habilidade em usar código como matéria prima para criar projetos de arte gerativa. Com um background em Design, mestre em Economia Criativa pela ESPM-Rio e atual doutorando em Artes Visuais no PPGAV-UFRJ, ele tem se destacado na comunidade de artistas-programadores, trazendo um olhar inovador para a interseção entre arte e tecnologia.

Glossário da Arte de Código Aberto

Arte de Código Aberto
Uma produção artística de base algorítmica que permite ser modificada desde o seu código-fonte, não apenas pelo artista criador, mas também pelo público.
Algoritmo
Conjunto de passos que um programa de computador executa para realizar uma tarefa específica.
Regras
Instruções que definem o funcionamento dos algoritmos.
Sistemas
Conjuntos de regras.
Sistemas Autônomos
Sistemas criados com regras cujo resultado não dependem da vontade de quem os criou.
Aleatoriedade
Regras com resultados imprevisíveis.
Arte Gerativa
Arte feita com sistemas autônomos. Geralmente, utiliza a aleatoriedade para criar.
Programação Criativa
Abordagem que utiliza código para criar expressões artísticas, visuais e interativas. Combinação de programação e arte para explorar novas formas de se expressar e questionar.
Grade (Grid)
Estrutura bidimensional composta por linhas e colunas que organiza elementos em uma grade. É frequentemente usado para posicionar e organizar objetos na tela.
Padrões
Arranjos visuais ou repetições organizadas de elementos. Em arte gerativa, padrões podem ser criados usando algoritmos para gerar composições visuais interessantes e complexas.
JavaScript
Linguagem de programação muito utilizada para criar páginas web interativas e dinâmicas.
p5.js
Uma biblioteca JavaScript de código aberto para criação de gráficos e interações visuais.
